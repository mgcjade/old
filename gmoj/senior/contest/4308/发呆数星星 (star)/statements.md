# 发呆数星星 (star)

## 题目描述



## 输入


          从文件 star.in 中读入数据。          一行两个整数 n,m。        

## 输出


          输出到文件 star.out 中。          一行一个整数表示答案。        

## 样例数据

### 样例输入 1

```
10 10

```

### 样例输出 1

```
12

```
### 样例输入 2

```
1000000000 10000000

```

### 样例输出 2

```
2560

```
